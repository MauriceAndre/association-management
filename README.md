# association-management (IN DEVELOPMENT)
>Manage your red cross association.

## Getting Started
### Installation
**Requirements**
- docker
<br>

#### Clone
```sh
$ git clone https://github.com/MauriceAndre/association-management.git
```

#### Setup
```sh
$ docker-compose up
```

## Running the tests
```sh
$ npm test
```
